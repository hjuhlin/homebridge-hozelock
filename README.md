# Homebridge Platform Aqua Temp Plugin
This is a plugin for Hozelock Cloud Controller.

# Support for
1.0.0
* Turn on and off water

# Default config
```json
"platforms": [
    {
        "name": "Hozelock Cloud Controller",
        "HubId": "[HubId]",
        "ControllerId": "[ControllerId]",
        "StandardDuration": 10,
        "UpdateTime": 60,
        "Debug": false,
    }
]
```
